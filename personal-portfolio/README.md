# Personal Portfolio - Next.js

Rohit Kota's portfolio built with Next.js 14, React 18, and Tailwind CSS.

## Getting Started

**Use Node 20** (required for compatibility):

```bash
nvm use 20   # if using nvm
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Build

```bash
npm run build
npm start
```
